# Image2Wallpaper
This app sets the picture as wallpaper with resizing, alignment and effect. 

This application sets a picture as a wallpaper with/without resizing.  Select a picture from the gallery, resize it, and set it as a wallpaper or images, convert it to PNG or JPEG

"By the way, you won’t find this app on the Play Store because it has been removed by Google Play. However, even though it’s no longer available there, you can still install the APK file. Don’t worry the app is safe, harmless, ad-free, small in size, and very useful for resizing your photos.
